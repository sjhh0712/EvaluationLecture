# 강의 평가 웹사이트   

Description   
-----------------------------------
<span>수강생들을 위한 과목 평가 커뮤니티 사이트로, 회원은 자신이 수강한 과목의 주관적 평가서를 작성하여 다른 회원들과 공유할 수 있다.</span>   
<span>회원은 다른 회원들의 강의 평가서를 추천하고, 원하는 강의평가를 검색하여 작성글을 볼 수 있다.</span>
